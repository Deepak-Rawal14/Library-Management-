# Library-Management-System

## Object-Oriented Features
- **Inheritance**: `Library` superclass → `PhysicalBook`, `ElectronicResource``
- **Polymorphism**: `LibraryMember` vs `Guest` (3-day access limit)
- **Encapsulation**: Resource cataloging & borrowing logic
- **BlueJ Implementation**

## Tech Stack
- Java (BlueJ IDE)
- OOP Design Principles

## Setup & Run
1. Clone: `git clone https://github.com/Deepak-Rawal14/Library-Management-.git`
2. Open in **BlueJ**
3. Right-click classes → **Compile**
4. Test: Catalog creation, borrowing, guest access
